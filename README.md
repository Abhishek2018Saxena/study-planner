📚 Smart Study Planner

A Smart Study Planner built using FastAPI and SQLite to help students manage their study tasks efficiently.

---

🚀 Features

- Add study tasks
- View all tasks
- Track progress (coming soon)
- Smart scheduling (coming soon)

---

🛠️ Tech Stack

- Python
- FastAPI
- SQLite
- SQLAlchemy

---

📂 Project Status

🚧 In Progress — Day 1 (Backend Setup Completed)

---

▶️ How to Run

1. Install dependencies
   
   pip install -r requirements.txt

2. Run the server
   
   uvicorn main:app --reload

3. Open in browser
   http://127.0.0.1:8000/docs
